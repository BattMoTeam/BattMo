Functional parameters

Battery models contain functional parameters. The most important one is probably the OCV function, but we have also the ionic conductivity and diffusion in the electrolyte. The function definitions can be entered in the input json file using one of the following format

  1. Function name

  2. Function formula

  3. Tabulated function

The json schema that describes the interface is available here.

The keyword to choose the format is given by functionFormat.

Let us go through each different format separatly. We start with en example for each

Function name

For a formula, the function format is "named function". With the key "functionName", we give the name of the function. In matlab, this function should then be found in the function path (see matlab doc). The key

Function formula

For a formula, the function format is "string expression". With the key "function", we give the expression which, once evaluated, will give the value of the function. With the key "argumentList", we declare the arguments of the function.
